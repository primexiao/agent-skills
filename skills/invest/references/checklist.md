# check — 买入前六关 Checklist

> 吸收自 xbtlin/ai-berkshire `investment-checklist.md` (MIT)。

对一个或多个标的快速过巴菲特买入前六关。目标是**排除坏选择**，不是找到最好的。支持多标的（逗号/空格分隔），多标的时最后必须输出对比总览表。

先做信息丰富度评级（A/B/C，见 `research.md` 前置）。C级公司"数据不足"≠"不通过"也≠"通过"，诚实标"灰色地带，需补充一手信息"，不要为填满表格硬凑。

数据按 `data-sources.md` 取（A股用 ashare_data.py），关键指标工具验算。

## 六关

**第一关：能力圈** — 能用一句话说清怎么赚钱吗？10年后大概率还在做什么？说不清赚钱方式 → 直接标"不在能力圈"，终止分析。★1-5。

**第二关：好生意（经济特征）** — 用数据说话：

| 指标 | 参考标准 |
|------|---------|
| ROE 5年均值 | >15%优秀，>20%卓越 |
| 毛利率 | >40%暗示定价权 |
| 自由现金流 | 持续为正、≈净利润 |
| 资本开支强度 | 轻资产优于重资产 |
| 负债 | 有息负债/净利润<3年 |

★5=全达标，★4=4项，★3=3项，★2=2项或趋势恶化，★1=多数不达标或FCF持续为负。

**第三关：护城河** — 五类逐项：品牌/定价权、转换成本、网络效应、规模优势、技术壁垒。每项给具体证据+变宽/变窄。追加：给对手100亿能复制吗？★1-5。

**第四关：管理层** — 诚实度（承诺vs交付）、资本配置记录、持股/薪酬一致性、所有者心态、治理（关联交易/商誉/审计）。严重诚信问题 = 硬性否决。★1-5。

**第五关：安全边际** — PE/前瞻PE/PB/股息率/FCF Yield + 历史分位；三情景估值（**工具计算**）。追加：判断错了最多亏多少？腰斩敢加仓吗？★1-5。

**第六关：决策纪律** — FOMO自查：因为别人推荐/最近涨得好才想买？停牌5年能接受吗？买入论述200字内写得清吗？

## 镜子测试（不可跳过）

> "我以 __元 买入 __，因为：1) 生意本质是__，我理解它；2) 护城河是__，在变宽/变窄；3) 管理层__；4) 当前价格相当于内在价值__折；5) 即使我错了，下行风险可控，因为__。"

**5句话说不完整 = 不买，没有例外。**

## 快速否决清单（触发任一条 → 直接否决）

- 说不清怎么赚钱
- 连续3年FCF为负且无改善迹象
- 管理层诚信污点
- 竞争优势被不可逆侵蚀
- 赚钱靠"下一个人出更高价"（博傻）
- 无法承受归零
- 买入理由 = "别人都在买"/"最近涨得好"
- 200字写不清买入理由

## 输出

每家公司：六关评分表 + 核心数据表 + 关键风险3-5条 + 镜子测试 + 明确结论（✅通过X/6关 / ❌未通过+哪条红线 / ❓灰色地带+争议点）。多标的附总览对比表。默认只在对话中输出；用户明确要求保存时，写入 `~/investing/reports/{标的}-checklist-{YYYYMMDD}.md`。
