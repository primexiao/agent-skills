# research — 四大师深度研究

> 吸收自 xbtlin/ai-berkshire `investment-research.md` (MIT)，适配单人使用。

对指定标的做巴菲特/芒格/段永平/李录四视角综合研究。适用于已上市公司；黄金/BTC 一律走 `macro.md`，禁止用本框架。

## 前置：信息丰富度评级（必须执行并写进报告开头）

| 等级 | 特征 | AI研究陷阱 | 应对 |
|------|------|-----------|------|
| A级 | 上市多年、覆盖密集 | 输出趋同市场共识，alpha有限 | 重点做反面检验：聪明人为什么不买？ |
| B级 | 上市1-3年、数据需推算 | 用"合理推测"伪装完整 | 每个推算数据标注置信度 |
| C级 | 冷门/新上市 | 把"资料少"误判为"不好" | 第一性原理：客户是谁？为什么付钱？复购靠什么？给对手100亿能复制吗？ |

偏见自查：我的"确定性"来自生意本质还是资料数量？结论是否与市场共识雷同？

## 第一步：数据收集

按 `data-sources.md` 规范取数（A股优先用 ashare_data.py）。需要覆盖：收入结构、近5年财务指标、竞争格局、护城河证据、管理层记录、TAM、风险因素、当前估值、多空双方论点。

数据量大时可派 1-2 个子 Agent 并行搜集（用 Agent 工具），但汇总时本人负责交叉验证。

**强制验证**（禁止心算，命令见 `data-sources.md`）：市值验算、收入/净利润/现金储备 cross-validate、估值指标 verify-valuation。工具输出嵌入报告附录"关键数据交叉验证记录"。工具报 ❌ 时必须排查原因才能继续。

易错点：市值币种单位（港币/人民币/美元亿）、FCF口径、债务是否含租赁、AB股经济权益≠投票权。

## 第二步：生意本质 — 段永平"对的生意"

一句话定义生意本质；收入结构拆解；5年盈利趋势；一次性销售vs复购；客户锁定强度；毛利率与同行对比（解释为什么高/低）；经营杠杆。
**追问**：这门生意好在哪？一句话说不清 = 没看懂。

## 第三步：护城河 — 巴菲特"经济护城河"

| 类型 | 验证方法 |
|------|---------|
| 品牌/定价权 | 提价不掉量？ |
| 转换成本 | 客户迁移成本多高？ |
| 网络效应 | 用户越多产品越好？ |
| 规模效应 | 成本优势多大？ |
| 技术/专利 | 领先几年？可复制？ |

过去5年变宽还是变窄？**追问**：10年后护城河还在吗？什么能摧毁它？

## 第四步：逆向思考 — 芒格"反过来想"

- "这家公司可能失败的所有路径"表（路径/概率/影响）
- 历史类比：相似位置的公司结局如何
- 偏误自查：叙事偏差、锚定、幸存者偏差
- 收集空方核心论点（真实空方，不是稻草人）

**追问**：我最可能在哪里犯错？聪明人为什么做空它？

## 第五步：管理层 — 段永平"对的人"

CEO关键决策复盘表（时间/决策/结果/评分）；资本配置能力（回购时机/并购成功率）；持股与薪酬结构；减持记录；关键人才风险。
**追问**：CEO退休后公司还能保持竞争力吗？

## 第六步：行业与长期趋势 — 李录

行业是否处于范式转移；TAM增长与天花板；产业链价值分配位置；技术路线风险；客户/供应商集中度。
**追问**：20年后回看，它是"这个时代的标准石油"还是"昙花一现的3Com"？

## 第七步：估值与安全边际

- 当前估值指标表（**工具验算**）
- 反向DCF：当前价隐含什么增长预期？
- 三情景估值（**必须用 three-scenario 工具**，禁止心算）
- 与自身历史、同行估值对比

**追问**：股市明天关闭5年，你愿意以这个价格持有吗？

## 第八步：综合决策备忘录

| 维度 | 结论 | 信心度 |
|------|------|--------|
| 生意质量（段永平） | | |
| 护城河（巴菲特） | | |
| 管理层 | | |
| 最大风险（芒格） | | |
| 长期趋势（李录） | | |

最终决策表：空仓者建议 / 持仓者建议 / 卖出信号 / 加仓信号。分层建议（激进/稳健/保守）+ 具体价格区间。

## 输出要求

1. 报告写入 `~/investing/reports/{标的}-research-{YYYYMMDD}.md`
2. 结论明确（买入/观望/回避 + 价格区间），不打太极
3. 报告开头：信息丰富度评级 + AI研究局限性声明
4. 报告结尾：区分"AI分析置信度"（取决于资料量）与"投资确定性"（取决于生意本质）
5. C级公司附"需要一手验证的问题清单"
6. 评分用★1-5，不含半星；估计值标"估计"；穿插四大师语录点评
