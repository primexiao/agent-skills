# gold / btc — 宏观资产分析

黄金和 BTC 没有现金流，**四大师价值投资框架不适用**：不存在内在价值锚，只有"宏观驱动 + 周期定位 + 相对历史区间"。本框架保留同样的反幻觉机制（双源验证、强制工具取数、强制结论），但必须显式承认：此类分析的置信度天然低于基于现金流的股票分析。

**硬性禁令**：报告中出现 PE、ROE、护城河、DCF 等企业估值词汇用于黄金/BTC 本体 → 打回重写。（矿业股/COIN/MSTR 等是股票，走 `research.md`。）

## 第一步：数据仪表盘（强制，先取数后分析）

```bash
python3 <skill_dir>/tools/macro_data.py dashboard   # 金价+BTC+DXY+实际利率；双源范围见 data-sources.md
```

输出原样嵌入报告。深度指标（央行购金/ETF流量/链上/COT）按 `data-sources.md` 宏观数据源表 WebSearch，**每个数据必须标注来源+数据日期**。

## 第二步：四视角分析（保留对抗结构，视角换成宏观口径）

### 视角一：宏观驱动（利率与流动性）

- **黄金**：10Y TIPS 实际利率水平与方向（长期负相关，但 2022 年后央行购金使金价对实际利率脱敏——先检验当前 regime 哪个主导，不要默认套用教科书关系）；美元指数；通胀预期（breakeven）
- **BTC**：全球 M2 / 美联储净流动性方向；实际利率；风险偏好（与纳指滚动相关性——BTC 是高 beta 流动性资产，不是"数字黄金"避险资产，用数据说话）

### 视角二：供需与筹码（谁在买，谁在卖）

- **黄金**：央行购金季度趋势（WGC）；黄金 ETF 流量；COT 投机净多头分位；金饰/工业需求
- **BTC**：现货 ETF 净流入（Farside，日度）；长期持有者供应占比变化；交易所余额趋势；稳定币总市值变化（场内弹药）

### 视角三：周期定位（现在在周期哪里）

- **黄金**：实际金价（经通胀调整）历史分位；金/M2、金/标普比值 vs 历史区间
- **BTC**：距上次减半的月数 + 历史同位置表现（**样本只有 4 次，必须写明 n=4、每轮减半边际影响递减的争议**）；MVRV Z-Score；距前高回撤 vs 历史回撤分布（-80%+ 发生过 3 次）

### 视角四：怀疑者（芒格式反过来想，强制执行）

- 当前市场主流叙事是什么？哪些可观察证据能**证伪**它？
- "这次不一样"检验：本轮与历史周期的结构性差异（ETF 化、主权购金、宏观 regime）是支撑还是陷阱？
- 黄金压力问题：实际利率若升 100bp 会怎样？央行购金停止/逆转的先行指标是什么？
- BTC 压力问题：ETF 净流出螺旋、监管黑天鹅、交易所/托管风险；"机构进场所以不会再暴跌"在 2022 年也说过

## 第四步：组合角色与仓位纪律（结论核心）

配置逻辑优先于择时逻辑：

先确认投资期限、流动性需求、现有敞口和最大可承受损失。用户未提供这些约束时，只给配置框架和待确认项，不给个性化仓位比例。

- **黄金 = 保险，不是收益资产**。以组合配置比和再平衡规则表达，不做频繁择时；任何示例区间都只是情景参数，不是默认建议
- **BTC = 高波动风险资产，仓位上限不能超过可承受 100% 归零的金额**。它不对冲任何东西；不得把通用百分比当成个性化建议
- 与现有股票持仓的相关性检查（BTC 与科技股正相关 → 加 BTC ≈ 加杠杆科技敞口）

**强制结论表**：

| 项目 | 结论 |
|------|------|
| 当前周期位置判断 | （附置信度） |
| 配置动作 | 约束完整时：增持/维持/减持至 X%；约束缺失时：待确认 |

**镜子测试（宏观版）**：
> "我配置 __% 的黄金/BTC，因为：1) 它在我组合里的角色是__；2) 当前周期位置是__（置信度__）；3) 主要驱动是__；4) 我错了的可观察标志是__；5) 错了最多亏__，我承受得起。"

5句话说不完整 = 不配置。
