# earnings — 财报精读

> 吸收自 xbtlin/ai-berkshire `earnings-review.md` (MIT)。

对 `{标的} {期间}` 做一手财报精读（如 `腾讯 2025Q4`、`茅台 最新`）。核心理念：读原文不读研报摘要——二手信息被筛选过、有时滞、缺语境。

## 前置：资料可得性评级

A级=拿到完整原文（10-K/年报/电话会纪要）→ 全流程；B级=部分原文 → 标注"非原始来源"；C级=只有新闻摘要 → 聚焦核心数据变化，跳过附注挖掘，标注"一手资料不足"。

## 第一步：获取一手资料

财报原文（A股：巨潮资讯；港股：披露易；美股：SEC EDGAR）+ 电话会纪要 + 致股东信。拿不到原文时按 `data-sources.md` 双源拼凑并标注。

## 第二步：核心数据提取与验证

- **利润表**：总收入及分部拆解、毛利率、经营利润率（区分GAAP/Non-GAAP）、净利润（剔非经常性）、EPS——逐项列 本期/上期/YoY/指引达标
- **现金流（巴菲特最看重）**：经营现金流/净利润比（>100%佳，<80%警惕）、资本开支构成（维护vs扩张）、FCF、回购分红、期末现金
- **资产负债表**：现金vs有息负债、应收账款周转天数变化（放松信用冲收入？）、存货周转变化（积压？）、商誉占比

关键数据用 `financial_rigor.py cross-validate / verify-market-cap / verify-valuation` 验证（命令见 `data-sources.md`）。

## 第三步：管理层讨论精读（MD&A）

**语气信号**：🟢坦诚（主动认问题+给具体原因）、🟢清晰（战略有量化目标）、🔴模糊（"我们相信""长期来看"）、🔴转移（避开问题谈别的）、🔴归因外部化（全怪宏观）。

**承诺追踪**：上期财报/电话会的具体承诺 vs 本期兑现（✅/❌/⚠️）。段永平："看管理层靠不靠谱，就看他以前说的话做到了没有。"

**Q&A质量**：提取分析师最尖锐的问题，评管理层回答质量（1-5）+是否回避。

按 `narrative-conflicts.md` 建立**投资叙事冲突表**，对照管理层叙事、财务报表、客户/渠道信号、竞争者表现与空方证据；指出解决最大分歧的下一期指标或事件。

## 第四步：附注与隐藏信息

必查：关联交易公允性、股权激励稀释、或有负债、会计政策变更、分部利润率差异、客户/供应商集中度。

异常信号：应收增速>收入增速（塞渠道）、存货增速>收入增速、经营现金流<净利润且扩大、资本化开支突增、非经常性收益占比上升。

## 第五步：历史对比

关键指标放进 ≥4个季度序列看趋势（利润率/增速/现金流质量/资本开支强度改善还是恶化）；与管理层此前指引对比偏差。

## 第六步：输出

结构：核心数据速览 → 本期最重要的3个变化（≤500字）→ 语气与承诺追踪 → 投资叙事冲突表 → 附注隐藏信息 → Q&A精选 → 与投资论文的关系（如有持仓）→ 结论。

结论必须明确：1) 超预期/符合/低于预期（禁止"基本符合"两面话）；2) 对投资论文：强化/无影响/削弱/破裂；3) 下一个催化剂；4) 已持有该加/持/减。

默认只在对话中输出；用户明确要求保存时，写入 `~/investing/reports/{标的}-earnings-{期间}.md`。

原则：读原文不读摘要；看变化不看绝对值；听语气不只听内容；查附注不只看正文；给结论不做汇总。
